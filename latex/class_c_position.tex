\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_c_position}{C\+Position}} class describes a specific position on the board game. The \mbox{\hyperlink{class_c_position}{C\+Position}} Object knows the size of the board (number of rows and columns). The size attribute may only be set by the \mbox{\hyperlink{class_c_board}{C\+Board}} class, while the board is being constructed.
